# WAIC 2026 OPC独立先锋挑战赛 - 一人公司AI自动化生产线Demo

## 项目概述

本项目展示AI驱动的"一人公司"新范式，通过构建自动化生产线实现轻量化创业。

### 三大环节

1. **上游：私有RAG知识库**
   - 基于IEEE TMC/TON顶刊论文
   - 构建个人智力资产库
   - 支持技术问答与路线规划

2. **中游：商业决策Agent**
   - 实时监控工业互联网招标信息
   - 智能匹配技术能力与项目需求
   - 自动生成竞标策略报告

3. **下游：IP运营Agent**
   - 自动生成短视频脚本
   - 一键生成技术专栏内容
   - 实现内容营销自动化

## 核心技术

- **E2eDRS算法** (IEEE TMC)：端到端深度强化学习确定性资源调度
- **TimeDRS算法** (IEEE TON)：时间相关DRL混合流调度
- **NGIT-TSN-S-I交换机**：信通院认证产品

## 性能指标

- 流量调度成功率提升 **60%**
- 决策时间缩短 **2.5倍**
- 可调度流数量提升 **1.33x~6.01x**
- 服务器负载均衡优化 **2.65x~3.87x**

## 快速启动

```bash
# 安装依赖
pip install -r requirements.txt

# 启动应用
run.bat
```

## 演示流程

1. 启动应用后，在浏览器中访问显示的本地地址
2. 在侧边栏选择不同环节进行演示
3. 上游环节：选择预设问题或输入自定义问题，查看RAG回答
4. 中游环节：查看招标信息，点击生成匹配报告
5. 下游环节：基于招标项目生成IP运营内容
